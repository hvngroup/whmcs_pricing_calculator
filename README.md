# whmcs_pricing_calculator
Pricing Calculator là một WHMCS addon module giúp quản trị viên tự động tính giá sản phẩm, configurable options và addons dựa trên giá Monthly của default currency. Module tự động tính giá cho tất cả billing cycles (Quarterly → Triennially) với discount tùy chỉnh, đồng thời quy đổi tỷ giá tự động
